# Count Down

A `while` loop needs three things: an initialization (set the starting value), a condition (decide whether to keep going), and an advance (move the value closer to ending). Get all three right and the loop runs exactly as many times as you intend.

## Your task

Write a function `count_down(n)` that uses a **`while` loop** to print the integers from `n` down to `1`, one per line.

## Requirements

- Use a `while` loop (not `for`).
- Print each integer on its own line, starting at `n` and ending at `1`.
- The function prints the values; it does not return anything.

## Example

```python
count_down(5)
```

Expected output:

```
5
4
3
2
1
```
