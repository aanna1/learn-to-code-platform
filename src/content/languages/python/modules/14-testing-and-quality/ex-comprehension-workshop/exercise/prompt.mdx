# Comprehension Workshop

List comprehensions let you build lists in a single readable line. They often replace a for-loop plus `.append()`, and they tend to read more like plain English than the loop version.

The general form is:

```python
[expression for item in iterable if condition]
```

The `if condition` part is optional — leave it out when you want every item.

## Your task

Implement three functions using list comprehensions. You may also use `map()` or `filter()` with `lambda` if you prefer, but a comprehension is the most idiomatic choice for each one.

### `squares_of_evens(numbers)`

Return a list of the squares of all even numbers in `numbers`.

```python
squares_of_evens([1, 2, 3, 4, 5, 6])  # [4, 16, 36]
squares_of_evens([1, 3, 5])            # []
```

### `word_lengths(words)`

Return a list of the lengths of each word in `words`, in the same order.

```python
word_lengths(["apple", "kiwi", "banana"])  # [5, 4, 6]
word_lengths([])                           # []
```

### `long_words(words, min_len)`

Return only the words from `words` whose length is greater than or equal to `min_len`.

```python
long_words(["apple", "kiwi", "banana", "fig"], 5)  # ["apple", "banana"]
long_words(["hi", "ok"], 5)                         # []
```

## Requirements

- All three functions must be implemented.
- Each must return a `list` (not a generator or other iterable).
- The order of elements in the returned list must match the order of the input.
