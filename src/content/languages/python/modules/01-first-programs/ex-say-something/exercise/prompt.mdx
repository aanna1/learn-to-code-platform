Python's `print()` function is your main tool for showing output. Every time you want your program to communicate something — a result, a message, a label — you'll reach for `print()`.

## Your task

Write three `print()` calls that produce **exactly** these three lines, in this order:

```
Hello, world!
I am learning Python.
Let's go!
```

Write your code inside the `main()` function in the starter — that's where the TODO comments are. When you press **Run**, `main()` is called automatically and you'll see your output. When it looks right, press **Submit** to check it against the hidden tests.

## Requirements

- The output must match the three lines above exactly, including capitalization and punctuation.
- Use three separate `print()` calls — one per line.
- Add a short comment above each `print()` call describing what it does.

<Callout type="tip">
Watch the apostrophe in `Let's go!`. If you wrap that string in single quotes, the apostrophe ends the string early. Use double quotes on the outside: `print("Let's go!")`.
</Callout>
