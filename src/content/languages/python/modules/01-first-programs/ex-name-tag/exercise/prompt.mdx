Variables let you store a value once and reuse it everywhere. Here you'll practice creating variables and printing their contents.

## Your task

Three variables have been started for you in the editor. Your job is to:

1. Assign a sensible value to each of `first_name`, `last_name`, and `city`.
2. Print each variable on its own line, in that order, so you can see your name tag when you press **Run**.

## Requirements

- `first_name` must be a non-empty string containing only a first name (no spaces).
- `last_name` must be a non-empty string containing only a last name (no spaces).
- `city` must be a non-empty string.
- Print each variable with `print(first_name)` — pass the variable, not a quoted literal like `print("Ada")`.

The hidden tests check the *values you assign to the variables*, so make sure each one holds a real value.

## Example

If you chose the values below, your output would look like this:

```
Ada
Lovelace
London
```

Your output will differ based on the values you choose — any valid names and city work.
