You're going to write two kinds of contracts: an abstract base class (ABC) that enforces a method exists and shares base-class behavior, and a Protocol that declares an interface without requiring inheritance.

## Part 1: Abstract base class

Write an abstract base class called `Storage` with:

- An abstract method `read(self, key: str) -> bytes` that subclasses must implement.
- An abstract method `write(self, key: str, value: bytes) -> None` that subclasses must implement.
- A concrete method `exists(self, key: str) -> bool` that calls `self.read(key)` and returns `True` if it succeeds without raising `KeyError`, and `False` otherwise. Do not mark `exists` as abstract — it should work for any subclass automatically.

Then write a concrete subclass `MemoryStorage(Storage)` that uses a plain dict to store bytes values. `read` should raise `KeyError` if the key is missing.

## Part 2: Protocol

Write a `Protocol` called `Serializable` with a single method `to_bytes(self) -> bytes`.

Write a function `save_all(items: list, store: Storage) -> None` that iterates over `items`, calls `.to_bytes()` on each item, and writes the result to `store` using a key like `"item_0"`, `"item_1"`, etc.

Then write a class `Message` with an attribute `text: str` that satisfies `Serializable` without inheriting from it. Its `to_bytes` method should return `self.text.encode()`.
