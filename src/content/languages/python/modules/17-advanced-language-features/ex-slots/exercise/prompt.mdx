In this exercise you'll work with `__slots__` directly and via `@dataclass(slots=True)`.

## Part 1: Slotted class

Write a class `Vector3` with three float attributes: `x`, `y`, and `z`. Use `__slots__` so that:

- Instances have no `__dict__`.
- Assigning an undeclared attribute raises `AttributeError`.
- A method `magnitude(self) -> float` returns `(x**2 + y**2 + z**2) ** 0.5`.
- A `__repr__` returns a string like `Vector3(1.0, 2.0, 3.0)`.

## Part 2: Slotted dataclass

Write a dataclass `Pixel` with integer attributes `r`, `g`, `b` (all default to `0`). Use `@dataclass(slots=True)` so instances have no `__dict__` and bad attributes are rejected.

## Part 3: Verifying the restrictions

Write a function `try_bad_attr(obj) -> str` that tries to set `obj.nonexistent = True` and returns `"blocked"` if `AttributeError` is raised, or `"allowed"` if it succeeds.
