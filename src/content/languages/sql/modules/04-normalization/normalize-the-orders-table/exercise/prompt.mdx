Time to normalize a real table by hand — the skill the exam actually tests. Here's the wide, 1NF table from the lecture, one row per product on an order:

```
order_items
+----------+---------+-----+--------+----------+-----------+-------------+-----------+-------------+
| order_id | product | qty | emp_id | emp_name | branch_id | branch_city | client_id | client_name |
+----------+---------+-----+--------+----------+-----------+-------------+-----------+-------------+
PK = (order_id, product)
```

Every cell is atomic, so it's in 1NF — but it's still carrying the redundancy you saw earlier: the employee's name, the branch's city, and the client's name repeat on every line. Work it down to 3NF and answer the four questions.
