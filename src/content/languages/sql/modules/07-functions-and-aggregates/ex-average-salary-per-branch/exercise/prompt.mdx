`AVG(salary)` over the whole table gives one company-wide number. The more useful question is usually "average salary *per branch*" — and that's what `GROUP BY` is for: it splits the rows into groups that share a value, then computes the aggregate once for each group.

The starter query still gives you one company-wide average.
