This is your first query — and you're not expected to know SQL yet. The point is only to see how this course's IDE answers a question: not with text in a terminal, but with a **table** in the results grid. Reading data back is the **R** in CRUD, and when it happens *without permission* it's the "confidentiality" breach from the lecture. Here, you have permission — so read away.

A small `students` table has already been created and filled for you (you can see it at the top of the editor). The starter query asks for just one column, `name`.

## Your task

Change the query so the results grid shows the **whole table** — every column (`id`, `name`, `major`) and every row.

Press **Run** to see the grid, then **Submit** to check it.

<Callout type="tip">
In SQL, `*` is shorthand for "every column." So `SELECT * FROM students;` means "give me every column, for every row, in the students table." That's the one change you need.
</Callout>
