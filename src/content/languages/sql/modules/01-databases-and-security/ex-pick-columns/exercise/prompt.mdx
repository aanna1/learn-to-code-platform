In the last exercise you asked for the whole table with `*`. But a query's real power is that *you* decide the shape of the table that comes back — you can ask for exactly the columns you want and nothing more. This is the everyday version of a serious security idea: only ever pull the data you actually need. A query that grabs `*` when it only needed a name is how extra, sensitive columns end up somewhere they shouldn't.

The same `students` table is set up for you. The starter query returns every column, including the `id`.
