# Say Something

Python's `print()` function is your main tool for showing output. Every time you want your program to communicate something — a result, a message, a label — you'll reach for `print()`.

## Your task

Write a program that prints **exactly** these three lines, in this order:

```
Hello, world!
I am learning Python.
Let's go!
```

Each line should come from a separate `print()` call.

## Requirements

- The output must match the three lines above exactly, including capitalization and punctuation.
- Use three separate `print()` calls — one per line.
- Add a comment above each `print()` call briefly describing what it does.

## Example output

```
Hello, world!
I am learning Python.
Let's go!
```
