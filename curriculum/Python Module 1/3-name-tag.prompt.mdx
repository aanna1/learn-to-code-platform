# Name Tag

Variables let you store a value once and reuse it everywhere. Here you'll practice creating variables and printing their contents.

## Your task

Three variables have been started for you in `starter.py`. Your job is to:

1. Assign appropriate values to each variable.
2. Print each variable on its own line, in this order: `first_name`, then `last_name`, then `city`.

## Requirements

- `first_name` must be a string containing only a first name (no spaces).
- `last_name` must be a string containing only a last name (no spaces).
- `city` must be a string containing a city name.
- Print each variable using `print()` — do not print string literals like `print("Ada")`. The test checks that you're using the variables.
- All three values must be non-empty strings.

## Example output

If you chose the values below, your output would look like this:

```
Ada
Lovelace
London
```

(Your output will differ based on the values you choose — any valid names and city work.)
