# Profile Card

A bare list of values isn't very readable. f-strings let you weave variables into sentences so the output actually communicates something.

## Your task

Three variables are provided for you — do not change their values. Use f-strings to produce this exact output (substituting the variable values, not hard-coded text):

```
Name: Ada Lovelace
Age: 36
Language: Python
Bio: Ada Lovelace has been coding in Python for years.
```

## Requirements

- Do **not** change the values of `first_name`, `last_name`, `age`, or `language`.
- Every `print()` call must use an f-string — no plain string concatenation.
- The `Bio:` line must reference `first_name`, `last_name`, and `language` — all three in a single f-string.
- Output must match the format above exactly, including the labels (`Name:`, `Age:`, etc.) and spacing.

## Hint

The `Name:` line combines two variables in one f-string: `first_name` and `last_name` with a space between them.
