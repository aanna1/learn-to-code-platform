# Slice It

Strings are sequences — every character has a numbered position, and you can pull out any piece you want using index and slice notation. This exercise drills exactly that: no methods, no loops, just you and square brackets.

## Your task

Write a program that starts with the string `"extraordinary"` and prints **exactly** these six lines using only indexing and slicing — no string methods, no hardcoded substrings like `"extra"`.

```
e
y
extra
ordinary
yranidroartxe
rdin
```

## Requirements

- Use the variable `word = "extraordinary"` as your starting point.
- Each print statement must use an index or slice expression on `word` — not a literal string.
- Line 1: the first character (index).
- Line 2: the last character (negative index).
- Line 3: the first five characters (slice).
- Line 4: everything from position 5 onward (slice).
- Line 5: the whole word reversed (`[::-1]`).
- Line 6: characters at positions 6 through 9 inclusive (slice).

## Example output

```
e
y
extra
ordinary
yranidroartxe
rdin
```
