You have two implementations of the same function. One is slow. Your job is to use `cProfile` to find the hot spot, compare implementations with `timeit`, and write the faster version yourself.

## Part 1: profile a workload

Write a function `profile_workload(fn, *args) -> str` that:

- Runs `fn(*args)` under `cProfile.Profile`.
- Returns the profiler stats as a string, sorted by cumulative time, showing the top 10 lines.

Use `pstats.Stats` with a `io.StringIO` stream to capture the output as a string rather than printing it. Here's the shape:

```python
import cProfile, pstats, io

profiler = cProfile.Profile()
profiler.enable()
fn(*args)
profiler.disable()

buf = io.StringIO()
pstats.Stats(profiler, stream=buf).sort_stats("cumulative").print_stats(10)
return buf.getvalue()
```
